## Getting Started

### Installation

1. Clone the repository with

```sh
git clone https://github.com/EnzoVieira/Ignite-ReactJS.git
```

2. Install packages with yarn

```sh
cd dtmoney/ && yarn
```

or with npm

```sh
cd dtmoney/ && npm install
```

## Usage

Once you already into `/dtmoney` folder, you can run it with yarn

```sh
yarn start
```

or npm

```sh
npm start
```
